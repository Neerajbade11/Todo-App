# Todo App

A minimalistic Todo application built using HTML, CSS, and JavaScript. Manage your tasks with ease!

## Features

- Add and delete tasks
- Mark tasks as completed
- Save tasks to local storage for persistence

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Font Awesome
